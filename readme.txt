Instructions to install:

The working project is in "innovation" folder

1. open terminal and cd to your favorite folder
2. git clone https://github.com/dpanov76/SimpleFruits.git
3. cd innovation
4. npm install
5. ng serve
6. ng test
7. look at http://localhost:4200 in chrome or any other browser
